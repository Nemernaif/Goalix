# KoraLive App Screens

## 1. Home Screen
- يعرض مباريات اليوم.
- تبويبات: مباشر، اليوم، غداً، الأمس.
- فلترة حسب الدوري.
- إظهار الفرق، الشعارات، الوقت، والنتيجة.
- زر لإضافة المباراة أو الفريق للمفضلة.

## 2. Match Details Screen
- اسم الفريقين والشعارات.
- النتيجة المباشرة.
- وقت المباراة.
- أحداث المباراة.
- الإحصائيات.
- التشكيلات.
- تقييم اللاعبين.
- ملخص YouTube بعد نهاية المباراة.

## 3. League Screen
- ترتيب الدوري.
- المباريات.
- النتائج.
- الهدافين.
- الصانعين.
- الفرق.

## 4. Team Screen
- معلومات الفريق.
- آخر النتائج.
- المباريات القادمة.
- قائمة اللاعبين.
- إحصائيات الفريق.

## 5. Player Screen
- معلومات اللاعب.
- الفريق.
- المركز.
- الأهداف.
- التمريرات الحاسمة.
- التقييم.
- آخر المباريات.

## 6. Favorites Screen
- الفرق المفضلة.
- البطولات المفضلة.
- إعدادات التنبيهات.

## 7. Highlights Screen
- ملخصات المباريات من YouTube.
- أهداف المباراة.
- فلترة حسب الدوري أو الفريق.
