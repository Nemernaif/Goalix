# Goalix Screen Wireframes

## 1. Splash Screen
- Logo: Goalix
- Tagline: Live Football Intelligence
- Dark premium background
- Neon green accent

## 2. Home Screen
Top:
- Goalix logo
- Search icon
- Notifications icon

Tabs:
- Live
- Today
- Tomorrow
- Yesterday

Content:
- Match cards grouped by league
- Each card shows:
  - Team logos
  - Team names
  - Score or match time
  - Match status
  - Favorite button

Bottom Navigation:
- Home
- Leagues
- Highlights
- Favorites
- Profile

## 3. Match Details Screen
Header:
- Home team
- Away team
- Score
- Match minute/status

Sections:
- Timeline
- Statistics
- Lineups
- Ratings
- Highlights
- AI Summary

## 4. League Screen
Header:
- League logo
- League name
- Country

Tabs:
- Matches
- Standings
- Top Scorers
- Top Assists
- Teams

## 5. Team Screen
Header:
- Team logo
- Team name
- Follow button

Sections:
- Upcoming matches
- Recent results
- Squad
- Statistics
- Highlights

## 6. Player Screen
Header:
- Player photo
- Player name
- Club
- Position

Stats:
- Goals
- Assists
- Rating
- Cards
- Minutes played

## 7. Highlights Screen
- Latest YouTube highlights
- Filter by league
