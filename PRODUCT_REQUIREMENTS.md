# KoraLive App

تطبيق كرة قدم احترافي يعرض النتائج المباشرة، الإحصائيات، الترتيب، الهدافين، الصانعين، تقييمات اللاعبين، وملخصات YouTube بعد المباراة.

## النسخة الأولى MVP

### 1. الصفحة الرئيسية
- مباريات اليوم
- مباريات مباشرة
- مباريات قادمة
- فلترة حسب الدوري
- فلترة حسب الفريق المفضل

### 2. صفحة المباراة
- النتيجة المباشرة
- وقت المباراة
- أحداث المباراة
- الأهداف
- الكروت
- التبديلات
- التشكيلة
- الإحصائيات
- تقييم اللاعبين
- رابط ملخص YouTube بعد نهاية المباراة

### 3. صفحة الدوري
- جدول الترتيب
- الهدافين
- الصانعين
- المباريات
- النتائج
- الفرق

### 4. صفحة الفريق
- المباريات القادمة
- آخر النتائج
- قائمة اللاعبين
- ترتيب الفريق
- أخبار الفريق

### 5. صفحة اللاعب
- الاسم
- المركز
- الفريق
- الأهداف
- التمريرات الحاسمة
- التقييم
- آخر المباريات

### 6. المفضلة
- اختيار الفرق المفضلة
- تنبيهات الأهداف
- تنبيهات بداية المباراة
- تنبيهات نهاية المباراة

## التقنيات المقترحة

- Mobile App: Flutter
- Backend: Node.js
- Database: Supabase أو Firebase
- Football Data API: SportMonks أو API-Football
- Video API: YouTube Data API
- Hosting: Render أو Railway
- Source Control: GitHub

## الهدف النهائي

تطبيق احترافي وسريع بتصميم حديث يجمع بين سرعة FotMob وعمق SofaScore.
